# Video-Based Heart Rate Estimation Using Facial Features

This project estimates heart rate using facial video captured from a webcam.
It uses remote photoplethysmography (rPPG), signal processing, and FFT.

## Features
- Face detection using Haar Cascade
- Forehead ROI selection
- Green channel signal extraction
- Butterworth bandpass filtering
- FFT-based heart rate estimation

## Tools Used
- MATLAB
- OpenCV
- Webcam

## Output
- Real-time heart rate in BPM
- Filtered signal and FFT plots
